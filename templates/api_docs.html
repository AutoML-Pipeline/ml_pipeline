Data Ingestion

Upload and manage datasets that will be used in the ML pipeline.

POST /data_ingestion
Upload Dataset

Upload and store a dataset file in MinIO storage.

Request

Content-Type: multipart/form-data

Parameter Type Required Description
file File Yes CSV, Parquet, or JSON file to upload
dataset_name String Yes Name for the dataset
description String No Description of the dataset
Response
{
    "dataset_id": "12345",
    "name": "example_dataset",
    "message": "Dataset uploaded successfully",
    "preview": {
        "num_rows": 1000,
        "num_columns": 10,
        "rows": [
            // First few rows of data as JSON
        ]
    }
}
GET /data_ingestion
List Datasets

Get a list of all available datasets.

Response
{
    "datasets": [
        {
            "id": "12345",
            "name": "example_dataset",
            "description": "Example dataset description",
            "file_type": "csv",
            "created_at": "2023-01-01T12:00:00",
            "num_rows": 1000,
            "num_columns": 10
            // Additional metadata
        },
        // More datasets
    ]
}

Get Dataset Info

Data Preprocessing

Clean, transform, and prepare data for machine learning.

POST /data_preprocessing
Preprocess Dataset

Perform preprocessing operations on a dataset.

Request

Content-Type: application/json

{
    "dataset_id": "12345",
    "operations": [
        {"type": "remove_nulls", "columns": ["col1", "col2"]},
        {"type": "fillna", "columns": ["col3"], "value": 0},
        {"type": "normalize", "columns": ["col4", "col5"], "method": "minmax"},
        {"type": "outlier_removal", "columns": ["col6"], "method": "iqr", "threshold": 1.5}
    ],
    "output_name": "preprocessed_dataset_name"
}
Response
{
    "job_id": "job-12345",
    "status": "COMPLETED",
    "processed_dataset_id": "67890",
    "message": "Data preprocessing completed successfully",
    "preview": {
        "num_rows": 950,
        "num_columns": 12,
        "rows": [
            // First few rows of processed data
        ]
    }
}
GET /data_preprocessing/operations
List Preprocessing Operations

Get a list of all available preprocessing operations.

Response
{
    "available_operations": {
        "remove_nulls": "Remove rows with null values in specified columns",
        "fillna": "Fill null values with specified value",
        "normalize": "Normalize values using specified method (minmax, zscore)",
        "outlier_removal": "Remove outliers using specified method (iqr, zscore)",
        "categorical_encoding": "Encode categorical variables (one-hot, label)",
        "datetime_features": "Extract features from datetime columns",
        "binning": "Bin continuous variables into discrete categories"
    }
}
GET /data_preprocessing/dataset/{dataset_id}/preview
Preview Dataset

Get a preview of the dataset (first few rows).

Path Parameters
Parameter Type Description
dataset_id String ID of the dataset
Query Parameters
Parameter Type Default Description
limit Integer 10 Number of rows to preview

Feature Engineering

Extract and transform features for ML models.

POST /feature_engineering
Engineer Features

Perform feature engineering operations on a dataset.

Request

Content-Type: application/json

{
    "dataset_id": "12345",
    "operations": [
        {"type": "polynomial_features", "columns": ["col1", "col2"], "degree": 2},
        {"type": "binning", "column": "col3", "bins": 5, "labels": ["v_low", "low", "medium", "high", "v_high"]},
        {"type": "text_vectorization", "column": "text_col", "method": "tfidf", "max_features": 1000},
        {"type": "pca", "columns": ["col4", "col5", "col6"], "n_components": 2}
    ],
    "output_name": "featured_dataset_name"
}
Response
{
    "job_id": "job-12345",
    "status": "COMPLETED",
    "processed_dataset_id": "67890",
    "message": "Feature engineering completed successfully",
    "preview": {
        "num_rows": 950,
        "num_columns": 15,
        "rows": [
            // First few rows of data with engineered features
        ]
    }
}
GET /feature_engineering/operations
List Feature Engineering Operations

Get a list of all available feature engineering operations.

GET /feature_engineering/dataset/{dataset_id}/features
Get Dataset Features

Get the list of features in the dataset.

Model Selection

Select the best ML model for your data.

GET /model_selection/models
List Available Models

Get a list of all available ML models for training.

POST /model_selection/recommend
Recommend Model

Get model recommendations based on dataset characteristics.

Request

Content-Type: application/json

{
    "dataset_id": "12345",
    "target_column": "target",
    "problem_type": "classification",
    "evaluation_metric": "accuracy"
}
POST /model_selection/compare
Compare Models

Compare multiple models on a dataset.

Request

Content-Type: application/json

{
    "dataset_id": "12345",
    "target_column": "target",
    "problem_type": "classification",
    "models": [
        {"name": "random_forest_classifier", "hyperparameters": {"n_estimators": 100}},
        {"name": "xgboost_classifier", "hyperparameters": {"n_estimators": 100}}
    ],
    "evaluation_metrics": ["accuracy", "f1", "precision", "recall"],
    "cv_folds": 5
}

Model Training & Evaluation

Train and evaluate machine learning models.

POST /model_training_evaluation/train
Train Model

Train a machine learning model on a dataset.

Request

Content-Type: application/json

{
    "dataset_id": "12345",
    "target_column": "target",
    "feature_columns": ["feature1", "feature2"],
    "model_name": "random_forest_classifier",
    "hyperparameters": {
        "n_estimators": 100,
        "max_depth": 10
    },
    "test_size": 0.2,
    "random_state": 42,
    "model_description": "Random Forest model for predicting X"
}
GET /model_training_evaluation/evaluate/{model_id}
Get Model Evaluation

Get evaluation metrics for a trained model.

GET /model_training_evaluation/models
List Trained Models

Get a list of all trained models.

POST /model_training_evaluation/cross_validate
Cross-Validate Model

Perform cross-validation for a model.

Visualization

Generate visualizations from data and models.

POST /visualization/data
Create Data Visualization

Generate visualizations from dataset data.

Request

Content-Type: application/json

{
    "dataset_id": "12345",
    "visualization_type": "histogram",
    "columns": ["column1"],
    "title": "Distribution of Values",
    "options": {
        "bins": 20,
        "figsize": [10, 6]
    }
}
POST /visualization/model/{model_id}
Create Model Visualization

Generate visualizations from model data.

Request

Content-Type: application/json

{
    "visualization_type": "confusion_matrix",
    "title": "Confusion Matrix",
    "options": {
        "figsize": [8, 8],
        "cmap": "Blues"
    }
}
GET /visualization/types
Get Visualization Types

Get available visualization types.

Deployment

Deploy trained models for prediction.

POST /deployment/deploy
Deploy Model

Deploy a trained ML model for predictions.

Request

Content-Type: application/json

{
    "model_id": "12345",
    "deployment_name": "my-model-endpoint",
    "description": "Deployed model for product recommendations",
    "version": "1.0"
}
GET /deployment/list
List Deployments

Get a list of all deployed models.

POST /deployment/{deployment_id}/predict
Predict

Make predictions using a deployed model.

Request

Content-Type: application/json

{
    "data": [
        {"feature1": 1.0, "feature2": 2.0},
        {"feature1": 3.0, "feature2": 4.0}
    ]
}
POST /deployment/{deployment_id}/batch_predict
Batch Predict

Make batch predictions using a deployed model.

Request

Content-Type: application/json

{
    "dataset_id": "12345",
    "output_dataset_name": "predictions_output"
}
